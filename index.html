TAGS (CONTAINERS):

The component used to design the structure of websites are called HTML tags.

1. HEADING TAG: used for heading. There are six heading tags tags having different sizes.In heading tag,the things that we dont want to display are written. Represented by head.

2. PARAGRAH TAG:To add paragarphs.Ignores the spaces and displays text in a paragraph form. Represented by p.

3. ANCHOR TAG: used to add links to pages for other websites. There are 2 types of links- Absolute Link (link of other website) and Relative link (link of another file from our collection). Represented as:

4. BODY TAG: Body tag is used to add content- heading , paragraphs.The things that we want to display on the webpage are written in body tag.

5. BR TAG: used to add next line(line breaks). It has no closing tag. Represented by br.

6. IMAGE TAG: used to add images. If by mistake the link is not correct, the image will not be loaded so alt attribute is used to diplay name of the image or any related information if in case image cant be loaded. Size can be altered by using attributes like height="" or width="", recommended to change one at a time.

7. BOLD, ITALIC, UNDERLINE TAGS: used to bold , underline to make the text italic.

8. BIG AND SMALL TEXT: used to show big or small tags.

9. HR TAG: used to display horizontal ruler,used to separate contents. It has no closing tag.

10. SUBSRIPT AND SUPERSCRIPT TAGS: can be used to write chemical formulas or mathematical formulas.

11. PRE TAG: used to display text as it is (without ignoring spaces and next line)

PAGE LAYOUT TECHNIQUES:

  • There are two types of tags - semantics tag and non semantic tags.
  • Semantic tags are those which can be understood by seeing them, include tags like header, footer, main.
  • Non semantics tags are those whose meaning can't be understood by seeing them, include tags like div, span, etc.

Semantic Tags: Semantic tags make the code more readable.Lines of code can be written in a more structured manner. Use of semantic tags is important as it improves the ranking. It is a good user experience.

Semantic tags include 3 tags - header, main, footer.

Main tag includes 3 tags inside it- section, main, footer.

1. Section Tag: For a section on your page.

2. Article Tag: For an article on your page.

3. Aside Tag: For content aside main content(ads).

LISTS IN HTML:

Lists are used to represent real life list data.

There are two types of lists: UNORDERED LIST and ORDERED LIST.

1. UNORDERED LIST: The data is printed in the form of bullets. li stands for list item.


2. ORDERED LIST: The data is printed in a sequence (numbers, roman numerals,alphabets).


TABLES IN HTML:

Tables are used to represent real life table data.

  • tr tag is used to diplay table row.
  • td tag is used to display table data.
  • th tag is used to display table header.
  • caption tag is used to add caption.
  • thead tag is used to wrap table head.
  • tbody tag is used to wrap table body.
  • colspan attribute is used to create cells which spans over multiple columns.

FORM IN HTML:

Forms are used to collect data from the user.

Example- Sign up/Login/help requests/contact me

  • Action in form: Action attribute is used to define what action needs to be performed when a form is submitted.
  • Form Element- Input
